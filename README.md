# dao-dir-monorepo

## Packages
### dao-dir webapp
The [dao dir webapp](https://github.com/0xreklaw/dao-dir-webapp/tree/0e922b72b3d6b0e7faaf266ffc8ebb7132e0f852) is the frontend for interacting with the DAO Directory

## Quickstart

### Install dependencies

```sh
yarn
```

### Build all packages

```sh
yarn build
```
